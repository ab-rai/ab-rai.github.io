ChatBlock - Decentralized Messaging App

Description: ChatBlock is a decentralized messaging web application developed to enable secure communication using blockchain technology. The application stores data in the blockchain in JSON format and adds digital fingerprints to the blocks using the SHA‑256 hashing function. Users can log in and send messages through a user-friendly interface built with ReactJS. The app uses hashing to implement the proof-of-work algorithm and the mining process for enhanced security.
